# Contributing to Tracebit

First — thank you. Every contribution 
makes Tracebit stronger for everyone.

---

## What we need help with

- Improving detection accuracy
- Adding new platform adapters
- Writing tests
- Improving documentation
- Translating docs to other languages
- Reporting bugs

---

## How to contribute

1. Fork this repository
2. Create a new branch for your change
3. Make your changes
4. Submit a pull request

---

## Rules

- Be respectful to all contributors
- Write clean readable code
- Comment your code clearly
- Test before submitting
- One feature per pull request

---

## Reporting bugs

Open an issue on GitHub and include:

- What you were doing
- What you expected to happen
- What actually happened
- Which browser and platform you were on

---

## Ideas and feature requests

Open a GitHub Discussion and describe
your idea clearly. We review every one.

---

## Code of conduct

Tracebit is open to everyone regardless
of background, experience level, or
location. Treat everyone with respect.
Harassment of any kind will not be
tolerated.

---

## Questions

Open a GitHub Discussion or reach out
directly through the repository.

Thank you for building Tracebit with us.
